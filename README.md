# منصة الخليج للخدمات اللوجستية

منصّة لوجستية عربية (RTL) للمملكة العربية السعودية: خدمات النقل والتخليص الجمركي والشحن والتخزين،
مع نظام إعلانات (نقل / تخليص / بيع شاحنات ومعدات)، لوحة تحكم للمستخدم، ولوحة إدارة كاملة.

> **واجهة أمامية فقط (Frontend only).** البيانات تُحفظ في `localStorage` داخل المتصفح،
> والتحقق عبر OTP محاكى محلياً (Authentica يحتاج خادماً). الكود مهيّأ ليسهل ربطه بخدمة حقيقية لاحقاً.

## التشغيل

```bash
npm install --legacy-peer-deps   # تثبيت الحزم
npm run dev                      # خادم التطوير على http://localhost:3000
npm run build                    # بناء الإنتاج
npm run lint                     # فحص ESLint
```

## الحسابات التجريبية

| الدور    | رقم الجوال     | كلمة المرور |
| -------- | -------------- | ----------- |
| مستخدم   | `0512345678`   | `user1234`  |
| مدير     | `0500000000`   | `admin123`  |

- **رمز التحقق (OTP) التجريبي:** `1234` (يُستخدم في صفحة تفعيل الحساب بعد التسجيل).
- لوحة الإدارة على المسار `/admin` (تتطلب حساب مدير).
- زر **إعادة تعيين البيانات التجريبية** متاح في لوحة الإدارة لاسترجاع الحالة الافتراضية.

## أبرز الميزات

- **المصادقة:** تسجيل (الاسم، جوال سعودي، كلمة المرور) ← تحقق OTP ← تسجيل دخول.
- **الرئيسية:** بانر رئيسي، بطاقات الخدمات الثلاث مع خياراتها الفرعية، روابط سريعة لمواقع الجهات،
  وأحدث إعلانات النقل/التخليص/البيع.
- **طلب خدمة:** نماذج ديناميكية (نقل / تخليص / شحن وتخزين) تُحفظ وتظهر في لوحة الإدارة ولوحة المستخدم.
- **إضافة إعلان:** نقل / تخليص / بيع (مع صور متعددة لإعلانات البيع).
- **صفحة تفاصيل الإعلان:** معرض صور + زرّا واتساب والاتصال.
- **لوحة المستخدم:** نظرة عامة، طلباتي، إعلاناتي، الملف الشخصي، تغيير كلمة المرور.
- **لوحة الإدارة:** إحصائيات + إدارة المستخدمين، البانرات، الروابط السريعة، أنواع الحمولات،
  طلبات الخدمة، والإعلانات.
- **تصميم متجاوب** مع شريط تنقّل سفلي للجوال ودعم كامل لاتجاه RTL.

## التقنيات

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS v4** (سمة كحلي/أزرق، خطوط Cairo / Tajawal)
- طبقة بيانات وهمية عبر React Context + `localStorage`

## بنية المشروع

```
app/
  (site)/            # الصفحات العامة (رئيسية، خدمات، إعلانات، تفاصيل، أضف إعلان، تواصل)
  auth/              # تسجيل دخول / حساب جديد / تحقق OTP
  dashboard/         # لوحة المستخدم
  admin/             # لوحة الإدارة
components/          # المكوّنات (layout, ui, forms, ads, home, admin)
lib/                 # types, constants, seed, utils, store/, auth/
```

## ملاحظات حول الإنتاج

- استبدل طبقة `lib/store` بواجهة برمجية حقيقية (REST/GraphQL) وقاعدة بيانات.
- استبدل OTP الوهمي في `lib/auth/AuthProvider.tsx` بتكامل **Authentica** عبر خادم.
- كلمات المرور هنا مخزّنة كنص صريح لأغراض العرض فقط — استخدم تجزئة آمنة (bcrypt) في الخادم.
